## Install Basic Apps

Run step 1:

```bash
sudo -E ./setup_fedora_1.sh
```

### Setup Zsh

Run:

```bash
./config_zsh.sh
```

```bash
zsh
```

### Install Ruby

Run:

```bash
./install_ruby.sh
```

### Install Elixir

Run:

```bash
./install_elixir.sh
```

### Reboot System

Run:

```bash
reboot
```

### Install Snap Packages

Run:

```bash
./install_snap_packages.sh
```

### Setup Docker

Run:

```bash
sudo ./config_docker.sh
```
