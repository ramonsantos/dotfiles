## Setup Fedora and Install Apps

Run step 1:

```bash
sudo -E ./setup_fedora_1.sh
```

Reboot the system and run step 2:

```bash
sudo -E ./setup_fedora_2.sh
```

### Setup Zsh

Run:

```bash
sudo -E ./config_zsh.sh
```

### Install Ruby

Run:

```bash
sudo -E ./install_ruby.sh
```

### Install Elixir

Run:

```bash
sudo -E ./install_elixir.sh
```
